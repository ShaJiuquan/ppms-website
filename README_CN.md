# 开特云科技官网

开特云科技有限公司官方网站 - 专业的物理测量系统解决方案提供商

## 🌟 网站特色

- **现代化设计** - 采用最新的 React 18 + TypeScript + Vite 技术栈
- **响应式布局** - 完美适配桌面端、平板和移动设备
- **流畅动画** - 精心设计的交互动画和视觉效果
- **SEO优化** - 完善的元数据和语义化标签

## 📦 技术栈

- **前端框架**: React 18
- **开发语言**: TypeScript
- **构建工具**: Vite 5
- **路由**: React Router v6
- **样式**: 原生 CSS (现代化渐变和动画)

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```
